    CNN en Keras#

    Como ya explicamos en la teor√≠a de redes neuronales convolucionales (CNNs), estas redes neuronales son muy √∫til cuando necesitamos trabajar con im√°genes.

    Teor√≠a CNNs

    CNN con im√°genes en blanco y negro#

    Para este caso vamos a usar el dataset de MNIST, que contiene im√°genes 28x28 en blanco y negro (1 canal de color) de n√∫meros escritos a mano.

    Cargar datos#

    import pandas as pd
import numpy as np

## Cargar datos de ejemplo
## MNIST es un dataset muy popular para entrenar con im√°genes
from tensorflow.keras.datasets import mnist

(x_train, y_train), (x_test, y_test) = mnist.load_data()

    Comprobar dataset y visualizar imagen#

    import matplotlib.pyplot as plt

x_train.shape # (60000, 28, 28) -> 60000 im√°genes de 28x28 p√≠xeles

single_image = x_train[0]
plt.imshow(single_image)

    Preprocesamiento de datos#

    ### En este caso queremos categorizar 10 valores (n√∫meros del 0 al 9).
from tensorflow.keras.utils import to_categorical

y_cat_test = to_categorical(y_test,10)
y_cat_train = to_categorical(y_train,10)


### Hay que verificar que los valores est√©n normalizados entre 0 y 1,
### normalmente nos viene en valores entre 0 y 255
single_image.max() # 255
single_image.min() # 0

x_train = x_train/255
x_test = x_test/255

scaled_single = x_train[0]
scaled_single.max() # 1


### Reformatear canales
### En este ejemplo nos falta el canal del color en el formato
### (n¬∫ im√°genes, ancho, alto, canales)
x_train.shape # (60000, 28, 28)

x_train = x_train.reshape(60000, 28, 28, 1)

    Preparar modelo#

    from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense, Conv2D, MaxPool2D, Flatten

model = Sequential()

# La primera capa siempre suele ser una capa convolucional
# CONVOLUTIONAL LAYER
### filters: A mayor n√∫mero de clasificaciones, mayor debe ser el n√∫mero de
###     filtros, suele ser un exponente de 2.
### kernel_size: Tama√±o de matriz donde vamos a aplicar el kernel/filtro.
###     Suele ser m√∫ltiplos de 2, empezar por (2,2) o (4,4) e ir subiendo.
### strides: Cu√°nto se mueve el kernel tras cada iteraci√≥n. En este ejemplo
###     se usa valor por defecto (1,1). Para im√°genes grandes es mejor aumentarlo.
### padding: Valor para decidir si queremos a√±adir padding o no. 
###     "valid" or "same" (default is "valid" == no padding).
### input_shape: Tama√±o imagen.
model.add(Conv2D(filters=32, kernel_size=(4,4),input_shape=(28, 28, 1), activation='relu',))

# POOLING LAYER
## Normalmente el par√°metro pool_size sigue la misma regla que kernel_size de la capa
##      convolucional.
model.add(MaxPool2D(pool_size=(2, 2)))

# FLATTEN IMAGES
# Aplanar im√°genes de 28x28 a 764 antes de la capa final
model.add(Flatten())

# Dense layer para disminuir el n√∫mero de neuronas, debe usarse un exponente de 2.
model.add(Dense(128, activation='relu'))

# La √∫ltima capa es la que se usa para clasificar, debe haber una neurona por clase.
# En etse caso tenemos 10 clases posibles.
model.add(Dense(10, activation='softmax'))

model.compile(loss='categorical_crossentropy',
              optimizer='adam',
              metrics=['accuracy']) # M√°s m√©tricas explicadas en https://keras.io/metrics/

    Entrenar modelo#

    from tensorflow.keras.callbacks import EarlyStopping

early_stop = EarlyStopping(monitor='val_loss',patience=2)

model.fit(x_train,y_cat_train,epochs=10,validation_data=(x_test,y_cat_test),callbacks=[early_stop])

    Evaluar modelo#

    # Mostrar m√©tricas del entrenamiento
model.metrics_names # ['loss', 'accuracy']

losses = pd.DataFrame(model.history.history)
losses[['accuracy','val_accuracy']].plot()
losses[['loss','val_loss']].plot()

model.evaluate(x_test,y_cat_test,verbose=0) # [0.04325260493150563, 0.9867]

# Reporte de la classificaci√≥n y matriz de confusi√≥n
from sklearn.metrics import classification_report,confusion_matrix

predictions = model.predict_classes(x_test)
predictions[0] #7
print(classification_report(y_test,predictions))

    CNN con im√°genes a color#

    Para este caso vamos a usar el dataset de CIFAR-10, que contiene im√°genes 32x32 a color (3 canales de colores - RGB) de 10 clases de objetos diferentes como aviones, gatos, perros, etc.

    La forma de trabajar con la red neuronal desde keras es 90% id√©ntica al ejemplo anterior cuando trabaj√°bamos con im√°genes en blanco y negro (1 canal de color).

    import pandas as pd
import numpy as np

# Cargar datos

(x_train, y_train), (x_test, y_test) = cifar10.load_data()

x_train.shape # (50000, 32, 32, 3) = 50000 im√°genes de 32x32 con 3 canales de colores
x_train[0].shape # (32, 32, 3)


# Normalizar im√°genes (dividir entre 255 cada pixel) es igual que antes
# (...)

# (...)


# Preparar modelo

from tensorflow.keras.layers import Dense, Conv2D, MaxPool2D, Flatten

model = Sequential()

# En este caso input_shape es diferente porque el tama√±o de la imagen es otro.
model.add(Conv2D(filters=32, kernel_size=(4,4),input_shape=(32, 32, 3), activation='relu',))

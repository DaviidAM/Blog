    CNN en Keras#

    Como ya explicamos en la teor칤a de redes neuronales convolucionales (CNNs), estas redes neuronales son muy 칰til cuando necesitamos trabajar con im치genes.

    Teor칤a CNNs

    CNN con im치genes en blanco y negro#

    Para este caso vamos a usar el dataset de MNIST, que contiene im치genes 28x28 en blanco y negro (1 canal de color) de n칰meros escritos a mano.

    Cargar datos#

    import pandas as pd

    Preprocesamiento de datos#

    ### En este caso queremos categorizar 10 valores (n칰meros del 0 al 9).

y_cat_test = to_categorical(y_test,10)

### Hay que verificar que los valores est칠n normalizados entre 0 y 1,
### normalmente nos viene en valores entre 0 y 255

### Reformatear canales
### En este ejemplo nos falta el canal del color en el formato
### (n췈 im치genes, ancho, alto, canales)

    Preparar modelo#

    from tensorflow.keras.models import Sequential
from tensorflow.keras.layers import Dense, Conv2D, MaxPool2D, Flatten

model = Sequential()

# La primera capa siempre suele ser una capa convolucional
# CONVOLUTIONAL LAYER
### filters: A mayor n칰mero de clasificaciones, mayor debe ser el n칰mero de
###     filtros, suele ser un exponente de 2.
### kernel_size: Tama침o de matriz donde vamos a aplicar el kernel/filtro.
###     Suele ser m칰ltiplos de 2, empezar por (2,2) o (4,4) e ir subiendo.
### strides: Cu치nto se mueve el kernel tras cada iteraci칩n. En este ejemplo
###     se usa valor por defecto (1,1). Para im치genes grandes es mejor aumentarlo.
### padding: Valor para decidir si queremos a침adir padding o no. 
###     "valid" or "same" (default is "valid" == no padding).
### input_shape: Tama침o imagen.
model.add(Conv2D(filters=32, kernel_size=(4,4),input_shape=(28, 28, 1), activation='relu',))

# POOLING LAYER
## Normalmente el par치metro pool_size sigue la misma regla que kernel_size de la capa
##      convolucional.
model.add(MaxPool2D(pool_size=(2, 2)))

# FLATTEN IMAGES
# Aplanar im치genes de 28x28 a 764 antes de la capa final
model.add(Flatten())

# Dense layer para disminuir el n칰mero de neuronas, debe usarse un exponente de 2.
model.add(Dense(128, activation='relu'))

# La 칰ltima capa es la que se usa para clasificar, debe haber una neurona por clase.
# En etse caso tenemos 10 clases posibles.
model.add(Dense(10, activation='softmax'))

model.compile(loss='categorical_crossentropy',
              optimizer='adam',
              metrics=['accuracy']) # M치s m칠tricas explicadas en https://keras.io/metrics/

    Entrenar modelo#

    from tensorflow.keras.callbacks import EarlyStopping

model.fit(x_train,y_cat_train,epochs=10,validation_data=(x_test,y_cat_test),callbacks=[early_stop])

    Evaluar modelo#

    # Mostrar m칠tricas del entrenamiento
